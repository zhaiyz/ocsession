# 搜索功能使用说明

## 🔍 搜索功能

### 如何使用

**1. 进入搜索模式**
- 按 `/` 键进入搜索模式
- 光标会显示在搜索框中

**2. 输入搜索词**
- 直接输入字符，实时过滤会话
- 支持中英文搜索
- 搜索范围：会话标题 + 目录路径

**3. 导航搜索结果**
- 使用 `j/k` 或 `↑/↓` 在结果中导航
- 右侧预览面板实时更新

**4. 确认或取消**
- 按 `Enter` 确认搜索，保持过滤结果
- 按 `Esc` 取消搜索，恢复完整列表

---

## ⌨️ 搜索模式快捷键

| 快捷键 | 功能 |
|--------|------|
| 字符输入 | 实时添加到搜索词 |
| `Backspace` | 删除最后一个字符 |
| `Esc` | 取消搜索，恢复列表 |
| `Enter` | 确认搜索，保持结果 |
| `j/k` 或 `↑/↓` | 在搜索结果中导航 |

## 🐛 常见问题

### Q: 为什么搜索没有结果？
**A**: 可能原因：
1. 搜索词拼写错误
2. 会话标题/目录不包含该词
3. 尝试简化搜索词

### Q: 如何清空搜索词？
**A**: 
- 方法1: 按 Backspace 逐个删除
- 方法2: 按 Esc 取消搜索

### Q: 搜索后如何恢复完整列表？
**A**: 按 Esc 取消搜索，或删除所有搜索字符
