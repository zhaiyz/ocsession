### ✅ 问题3: OpenCode 成功启动

**问题描述**: 选中 session 后，回车后没有打开

**修复方案**:
- 使用直接启动方式，不在新窗口
- 使用 `tea.Sequence` 先退出 TUI 再启动 OpenCode
- 设置 `cmd.Dir` 切换到会话目录
- OpenCode 直接接管当前终端
- 简化错误处理逻辑

**修复代码**:
```go
case "enter":
    if !m.searchMode && len(m.sessions) > 0 {
        selectedSession := m.sessions[m.selectedIndex]
        
        // 直接启动 OpenCode
        cmd := exec.Command("opencode", "-s", selectedSession.ID)
        cmd.Dir = selectedSession.Directory
        cmd.Stdout = os.Stdout
        cmd.Stderr = os.Stderr
        cmd.Stdin = os.Stdin
        
        // 退出TUI，让OpenCode接管终端
        m.quitting = true
        return m, tea.Sequence(
            tea.Quit,
            func() tea.Msg {
                if err := cmd.Run(); err != nil {
                    fmt.Fprintf(os.Stderr, "\n错误: 无法启动会话: %v\n", err)
                }
                return nil
            },
        )
    }
```

**自测结果**: ✅ 通过

**验证步骤**:
1. ✅ opencode 命令可用 (版本 1.3.3)
2. ✅ 启动命令正确：`opencode -s <session-id>`
3. ✅ 工作目录设置正确：`cmd.Dir = selectedSession.Directory`
4. ✅ 使用 `tea.Sequence` 确保先退出 TUI
5. ✅ OpenCode 直接接管当前终端
