### ✅ 问题2: 标题乱码

**修复前**：
- 中文字符宽度计算不准确
- 标题可能重叠或显示不全
- 特殊字符（emoji、符号）显示异常

**修复后**：
- 使用 `runewidth` 正确计算中文字符显示宽度
- 标题根据实际显示宽度截断
- 避免字符重叠和乱码

**验证方法**：
1. 启动应用：`./bin/ocsession`
2. 检查包含中文的标题是否正常显示
3. 检查右侧预览面板的标题是否正确
4. 长标题应该正确截断显示 "..."

## 🎨 改进细节

### 时间格式改进

**之前**：
- 今天：`15:04`
- 本周：`周一 15:04`（中文星期）
- 更早：`2006-01-02`

**现在**：
- 今天：`今天 15:04`（明确标识）
- 本周：`Mon 15:04`（英文星期，固定宽度）
- 更早：`2006-01-02`（固定宽度）

### 标题处理改进

**之前**：
```go
func truncate(s string, maxLen int) string {
    if len(s) <= maxLen {  // 按字节长度判断
        return s
    }
    return s[:maxLen-3] + "..."  // 可能截断中文字符
}
```

**现在**：
```go
func truncate(s string, maxLen int) string {
    width := runewidth.StringWidth(s)  // 计算显示宽度
    if width <= maxLen {
        return s
    }
    return runewidth.Truncate(s, maxLen-3, "...")  // 正确截断
}
```
